# Habit Tracker Web App

A full-stack social habit tracking application that helps users build consistent habits while staying accountable through a community of friends.

## 🌐 Live Demo

- **Frontend**: https://habit-tracker0.netlify.app/
- **Backend API**: https://habit-tracker-gx4l.onrender.com/api

## ✨ Features

- 🔐 **User Authentication** - Secure registration and login
- 📝 **Habit Management** - Create, edit, and delete habits with custom frequency
- 📊 **Progress Tracking** - Daily/weekly check-ins with streak counters
- 👥 **Social Accountability** - Follow friends and view their progress
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile

## 🛠 Tech Stack

**Frontend**
- React
- Tailwind CSS
- Axios
- React Router

**Backend**
- Node.js
- Express.js
- JWT Authentication
- bcrypt

**Database**
- MongoDB
- Mongoose ODM

**Deployment**
- Netlify (Frontend)
- Render (Backend)
- MongoDB Atlas (Database)
